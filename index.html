ABOUT THE PROJECT

About the project

Given the importance of covers as a vehicle through which to self-advertise the content itself and the chance to read books on different devices, in addition to the classic book. I thought it would be interesting to create a css library that would allow to adapt the cover to multiple formats in an easy and fast way. In particular the library is composed of an html file that contains all the information that must be present on a cover and four css style sheets, one for each format: tablet, smartphone, traditional cover and Ereader. So it is possible to link the desired one to the reference html file.

In order to show how the library works, I decided to recreate the cover of the text of Alessandro Manzoni's: I promessi sposi (The Betrothed) trying to make it more modern and attractive while keeping it simple both to be different from the covers currently on the market and to attract the attention of a young audience that has to study it at school and for lovers of Italian literature.

How to use it

Connect the css file to the html file and make changes to the files, if necessary, in order to create the cover page.


	< link rel="stylesheet" type="text/css" href="stylesheet-book.css ">
	
OR

	< link rel="stylesheet" type="text/css" href="stylesheet-smartphone.css ">
	
OR

	< link rel="stylesheet" type="text/css" href="stylesheet-tablet.css ">
	
OR

	< link rel="stylesheet" type="text/css" href="stylesheet-ereader.css ">

To conclude

Through this thesis project, born with the intent to analyze the book covers, as advertising space, both from the historical and graphic point of view, in this case through the analysis of three publishing houses such as Adelphi, Einaudi and Il Mulino; I wanted to use html and css in an alternative way to the classical uses, that is to create the book covers in different formats in order to facilitate their realization.
